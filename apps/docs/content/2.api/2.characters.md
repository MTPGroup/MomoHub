## 获取公开角色列表

::endpoint{method="GET" path="/characters/public"}
获取所有公开的 AI 角色列表，支持分页和搜索。
::

**查询参数：**

::field{name="page" type="number"}
页码，默认 1
::

::field{name="limit" type="number"}
每页数量，默认 20
::

::field{name="search" type="string"}
搜索关键词
::

**响应数据：** `PagedResponse<CharacterResponse>`

---

## 创建角色

::endpoint{method="POST" path="/characters" auth}
创建一个新的 AI 角色。
::

**请求参数：**

::field{name="name" type="string" required}
角色名称
::

::field{name="avatar" type="string"}
头像 URL
::

::field{name="bio" type="string"}
角色简介
::

::field{name="originPrompt" type="string"}
角色系统提示词
::

::field{name="isPublic" type="boolean"}
是否公开，默认 `false`
::

**请求示例：**

```bash
curl -X POST https://api.momohub.com/v1/characters \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "智能助手",
    "bio": "一个友好的 AI 助手",
    "originPrompt": "你是一个乐于助人的 AI 助手，用中文回答问题。",
    "isPublic": true
  }'
```
