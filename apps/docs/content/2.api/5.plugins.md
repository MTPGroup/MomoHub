## 获取插件列表

::endpoint{method="GET" path="/plugins"}
获取所有可用的插件列表，支持分页。
::

**查询参数：**

::field{name="page" type="number"}
页码，默认 1
::

::field{name="limit" type="number"}
每页数量，默认 20
::

**响应数据：** `PagedResponse<PluginResponse>`

```json
{
  "items": [
    {
      "id": "plg_abc123",
      "name": "Web Search",
      "description": "在线搜索插件，支持实时网络信息检索",
      "version": "1.0.0",
      "schema": {
        "name": "web_search",
        "description": "Search the web",
        "parameters": {
          "type": "object",
          "properties": {
            "query": { "type": "string" }
          }
        }
      },
      "authorId": "usr_abc123",
      "status": "active",
      "likeCount": 42,
      "createdAt": "2025-01-01T00:00:00Z",
      "updatedAt": "2025-01-15T08:30:00Z"
    }
  ],
  "total": 1,
  "page": 1,
  "limit": 20,
  "totalPages": 1,
  "hasNext": false,
  "hasPrevious": false
}
```

## 获取插件详情

::endpoint{method="GET" path="/plugins/:id"}
根据 ID 获取插件详细信息，包含源代码。
::

**响应数据：**

```json
{
  "id": "plg_abc123",
  "name": "Web Search",
  "description": "在线搜索插件",
  "version": "1.0.0",
  "schema": {
    "name": "web_search",
    "description": "Search the web",
    "parameters": { ... }
  },
  "code": "export default async function(params) { ... }",
  "authorId": "usr_abc123",
  "status": "active",
  "likeCount": 42,
  "createdAt": "2025-01-01T00:00:00Z",
  "updatedAt": "2025-01-15T08:30:00Z"
}
```

> 与列表接口相比，详情接口会额外返回 `code` 字段，包含插件的完整源代码。
